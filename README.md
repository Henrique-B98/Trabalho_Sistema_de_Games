## Requisitos Técnicos
- Herança:
  - Superclasse Cliente
  - Subclasses ClienteRegular, ClientePremium
    
- Polimorfismo:
  - Método validarSaldo() e exibirInfo()
  - Varia entre os tipos de clientes

- Encapsulamento:
  - Presente na classe Cliente
  - Por meio de atributos Private e uso de Getters e Setters
 
- Uso de Try-Catch em validarSaldo()
- Tratamento de exceções em validarSaldo()
